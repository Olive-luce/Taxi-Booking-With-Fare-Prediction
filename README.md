# Taxi-Booking-With-Fare-Prediction

Taxi Booking with Fare Prediction is a web-based application built using PHP for the frontend and MySQL as the database. It allows users to book taxis easily while providing an estimated fare based on factors such as distance, time, and route. The system stores user, driver, and trip data efficiently, ensuring smooth booking management and quick access to ride history. The fare prediction feature enhances user experience by offering transparency and helping customers make informed travel decisions.
